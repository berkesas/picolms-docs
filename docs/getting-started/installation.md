# Installation

Installing Pico is easy.

```bash
npm install @scinforma/picolms
```

or with yarn:

```bash
yarn add @scinforma/picolms
```

or with pnpm:

```bash
pnpm add @scinforma/picolms
```

## Peer Dependencies

This package requires React 18 or higher:

```bash
npm install react react-dom
```
